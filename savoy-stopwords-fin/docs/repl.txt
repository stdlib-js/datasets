
{{alias}}()
    Returns a list of Finnish stop words.

    Returns
    -------
    out: Array<string>
        List of stop words.

    Examples
    --------
    > var list = {{alias}}()
    [ 'aiemmin', 'aika', 'aikaa', ... ]
