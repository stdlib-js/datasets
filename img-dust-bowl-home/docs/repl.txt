
{{alias}}()
    Returns a `Buffer` containing image data of Dorothea Lange's gelatin silver
    print of an abandoned Dust Bowl home.

    Returns
    -------
    out: Buffer
        Image data.

    Examples
    --------
    > var img = {{alias}}()
    <Buffer>
