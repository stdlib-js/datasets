
{{alias}}()
    Returns a list of common female first names in English speaking countries.

    Returns
    -------
    out: Array<string>
        List of common female first names.

    Examples
    --------
    > var list = {{alias}}()
    [ 'Aaren', 'Aarika', 'Abagael', 'Abagail', ... ]
