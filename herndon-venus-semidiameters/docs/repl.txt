
{{alias}}()
    Returns fifteen observations of the vertical semidiameter of Venus, made by
    Lieutenant Herndon, with the meridian circle at Washington, in the year
    1846.

    This dataset is a classic dataset commonly used in examples demonstrating
    outlier detection.

    Returns
    -------
    out: Array<number>
        Observations.

    Examples
    --------
    > var d = {{alias}}()
    [ -0.30, -0.44, ..., 0.39, 0.10 ]
