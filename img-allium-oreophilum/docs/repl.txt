
{{alias}}()
    Returns a `Buffer` containing image data of Karl Blossfeldt's gelatin silver
    print *Allium ostrowskianum*.

    Returns
    -------
    out: Buffer
        Image data.

    Examples
    --------
    > var img = {{alias}}()
    <Buffer>
