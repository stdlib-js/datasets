
{{alias}}()
    Returns a list of simple American-English words (revised Spache).

    Returns
    -------
    out: Array<string>
        List of simple American-English words.

    Examples
    --------
    > var list = {{alias}}()
    [ 'a', 'able', 'about', 'above', ... ]
